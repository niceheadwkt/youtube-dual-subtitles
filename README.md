# youtube-dual-subtitles

這是一個適用於 Google Chrome 瀏覽器的輕量化擴充功能 (Chrome Extension - Manifest V3)，能於 YouTube 播放影片時，自動將原生字幕翻譯成您指定的第二語言，並雙語同步顯示於播放器中。

---

## 🌟 主要功能與特色

*   **雙語對照**：在 YouTube 原生字幕下方，同步渲染高可讀性的第二字幕。
*   **多國語言設定**：支援繁體中文、簡體中文、英文、日文、韓文、西班牙文、法語、德語、越南語、泰語等多國語言。
*   **精美懸浮面板 (Popup UI)**：採用現代極簡深色系與玻璃擬態 (Glassmorphism) 設計，可即時調整：
    *   **第二字幕字體大小** (14px - 36px)
    *   **字幕背景深淺** (透明度 0% - 100%)
*   **高效快取機制**：內建翻譯快取 (Cache)，重複的台詞不重複請求 API，字幕顯示毫無延遲，且節省網路資源。
*   **免 API 金鑰**：預設介接免金鑰的 Google 翻譯服務，安裝即可立即使用。
*   **安全性優化**：透過 Service Worker (背景腳本) 轉發請求，完美繞過 Chrome MV3 的 CORS 跨網域限制，安全穩定。

## 📂 專案檔案結構

移轉到其他電腦時，請確保以下檔案皆在同一個資料夾中：
```text
YouTube-Dual-Subtitles/
├── manifest.json       # 擴充功能設定檔 (定義權限、腳本入口)
├── background.js       # 背景服務腳本 (繞過 CORS 處理翻譯請求)
├── content.js          # 網頁核心腳本 (偵測 YouTube 字幕與渲染)
├── style.css           # 網頁樣式檔 (調整第二字幕定位與字體陰影)
├── popup.html          # 設定面板介面 (HTML 結構)
├── popup.css           # 設定面板樣式 (CSS 美化)
└── popup.js            # 設定面板邏輯 (儲存設定與即時同步)
```

## 💻 另一台電腦安裝與設定步驟

要在您的新筆電 (NB) 上安裝此擴充功能，請按照以下詳細步驟操作：

### 第一步：準備專案資料夾
1. 將本專案資料夾（包含上述所有檔案）複製到新筆電的任意目錄中（例如放在 `D:\my-extensions\YouTube-Dual-Subtitles` 或是桌面）。

### 第二步：開啟 Chrome 擴充功能頁面
1. 在新筆電開啟 Google Chrome 瀏覽器。
2. 在網址列輸入 `chrome://extensions/` 並按下 **Enter**。

### 第三步：開啟「開發人員模式」
1. 在擴充功能管理頁面的**右上角**，找到 **「開發人員模式」(Developer mode)** 切換開關。
2. 將此開關**切換為開啟狀態**。

### 第四步：載入擴充功能
1. 點擊頁面左上角的 **「載入未封裝項目」(Load unpacked)** 按鈕。
2. 在彈出的資料夾選擇視窗中，瀏覽並**選擇剛才存放本專案的資料夾**。
3. 載入成功後，管理頁面上將會出現 **「YouTube 雙語字幕助手 (YouTube Dual Subtitles)」** 的卡片。

### 第五步：釘選擴充功能（方便使用）
1. 在瀏覽器右上角網址列旁，點擊 **「擴充功能」(拼圖圖示 🧩)**。
2. 在選單中找到 **「YouTube 雙語字幕助手」**。
3. 點擊旁邊的 **「圖釘 📌」** 圖示，將其固定在瀏覽器工具列上。

## 🎬 如何使用與測試

1. 開啟 [YouTube](https://www.youtube.com)，播放任何一部**有內建字幕（例如英文）**的影片。
2. 點擊 YouTube 播放器右下角的 **「CC (字幕)」** 圖示，開啟 YouTube 的原生字幕。
3. 點擊瀏覽器右上角剛才釘選的 **雙語字幕助手圖示**（紅色 YouTube 播放鍵圖示）。
4. 在面板中：
   * 確認 **「啟用雙語字幕」** 開關為開啟狀態。
   * 在 **「目標翻譯語言」** 下拉選單選擇您想顯示的語言（例如：繁體中文）。
   * 拖曳下方的滑桿來微調字體大小與背景透明度。
5. 面板設定會即時保存並套用，您將能同時看見原始字幕與翻譯後的第二字幕！

## 🔧 常見問題排除 (Troubleshooting)

*   **問題一：原字幕有出現，但第二字幕沒有出來？**
    *   *排查方式*：請確認您的網路連線正常（若 Google 服務受阻可能無法翻譯）。另外可前往 `chrome://extensions/`，找到此卡片並點擊右下角的 **「重新載入 🔄」** 鍵，再重新整理 YouTube 影片網頁試試。
*   **問題二：原生字幕關閉了，第二字幕還會顯示嗎？**
    *   *答*：不會。本功能是藉由監聽 YouTube 的原生字幕更新來運作，因此**必須啟用 YouTube 的原生字幕**，第二字幕才會同步翻譯顯示。
